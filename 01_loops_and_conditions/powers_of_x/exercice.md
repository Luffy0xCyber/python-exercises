# Exercise: Powers of X


## Instructions

1. Create a file named `power_of_x.py`.
2. Ask the user for a number (X) and a maximum exponent (N).
3. Validate that both numbers are strictly positive (> 0). If not, ask again.
4. Display a table showing the powers of X from exponent 0 up to N.
5. Match the formatting shown in the execution example.
