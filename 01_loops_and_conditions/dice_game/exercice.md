# Exercise: Dice Roll Simulation


## Instructions

1. Create a program named `de.py`.
2. Ask the user to choose a number between 1 and 6.
3. Validate the input. If the number is not between 1 and 6, ask again until a valid number is given.
4. Simulate dice rolls using `random.randint`.
5. Print the value of each roll.
6. Stop when the roll matches the user's number.
7. Display the total number of rolls at the end.
